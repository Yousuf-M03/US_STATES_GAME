# U.S. States Game

An interactive geography quiz built with Python, Turtle, and Pandas. Test your knowledge of U.S. state names by clicking on their correct locations on a map.

## Features
- Displays a blank U.S. map
- Prompts user to guess state names
- Tracks correct guesses (e.g., “12/50 States Correct”)
- Saves missed states to `states_to_learn.csv` for review
- Uses real coordinate data from `50_states.csv`

## Requirements
- Python 3.x
- `pandas`
- `turtle`
